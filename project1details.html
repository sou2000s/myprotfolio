It have google authenctication. Here a user can register him/her as user or a seller . When a user register user name ,email , and role saved in database.

It have jwt token authorization. If the user dont have authorization to acces the api or the route it will give a responose to the user "unauthorized access".

It have private , admin , seller route after verification then the user can access routes according to their role.

Sellers can delete, advertise and saw product status of his/her. When a products is sold out automaticly remove from advertice section and also seller will saw status sold.

Admin can delete users, sellers and reported item from his/her dashboard.

Buyrs can pay with stripe and report a product.
