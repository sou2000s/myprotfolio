My review page

It is travel service website.It have google authenctication. Here a user can register him/her and then able to add reviews for a package.

It have jwt token authorization. If the user dont have authorization to acces the api or the route it will give a responose to the user "unauthorized access".

Click on show all button user can see all packages .

User can add new package.

User can delete and edit their comments in my review tab.

Clicking on details button redirect the user to the specific package details page.

Technolodies: React js , tailwind css , express js , node js , vercel api hosting , jwt , firebaase authentication
